2. **Demo redesign:** "Summarise email" → "Log tonight's talk in your memory" — produces visible Discord file path. See [`demo-redesign-slide7.md`](demo-redesign-slide7.md).
3. **Foxconn claim:** "85% across all manufacturing" is wrong — it's two Taiwan battery plants. See [`claim_verification_results.md`](claim_verification_results.md).
4. **YouTube misattribution:** The "Seven Levels" video is by **Chase AI**, not Cole Medin. See [`youtube_video_verification_report.md`](youtube_video_verification_report.md).
5. **Two critical transitions:** 7→8 and 13→14 are weak. Replacement lines in [`TRANSITION-FIXES.md`](TRANSITION-FIXES.md).
